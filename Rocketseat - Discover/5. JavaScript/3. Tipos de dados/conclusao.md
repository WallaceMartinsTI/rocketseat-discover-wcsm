# Tipos de dados

Conforme o ECMAScript standard temos 9 tipos de dados:

* Data Types
    * Primitive / Primitive value
    * Structutal
    * Structutal Primitive

## Primitivos

* String
* Number
* Boolean
* undefined
* Symbol
* BigInt

## Estruturais

* Object
    * Array
    * Map
    * Set
    * Date
    * ...
* Function

## Primitivo Estrutural / Structutal Root Primitive

* null
